# Manual do StepAlbum

---

## Como importar álbuns da Kaz

### Passo 1 — Instale o bookmarklet (só precisa fazer uma vez)

1. Abra o StepAlbum e vá em **Álbuns → Importar da Kaz → Instalar bookmarklet**
2. Clique em **Copiar código**
3. Crie um favorito no seu navegador:
   - **Nome:** Kaz → StepAlbum
   - **URL/Endereço:** cole o código que você copiou
4. Salve. O favorito vai aparecer na barra de favoritos do navegador.

> **Como criar o favorito no Chrome:** clique com botão direito na barra de favoritos → "Adicionar página" → no campo URL, apague o que tiver e cole o código.

---

### Passo 2 — Copie os álbuns da Kaz

1. Abra o site da Kaz normalmente e faça login
2. Vá para a página de **Diagramações**
3. Clique no favorito **Kaz → StepAlbum** que você criou
4. Aguarde 1 ou 2 segundos (ele carrega todos os álbuns automaticamente)
5. Uma mensagem verde vai aparecer: **"✅ X álbuns copiados!"**

---

### Passo 3 — Importe no StepAlbum

1. Volte para o StepAlbum e clique em **Álbuns**
2. Clique no botão **Importar da Kaz**
3. Cole o conteúdo com **Ctrl+V** (ou Cmd+V no Mac)
4. Clique em **Processar**
5. Revise a lista de álbuns que aparece:
   - Escolha o **Tipo** de cada álbum (Faculdade, Medicina, etc.)
   - Escolha o **Responsável** — qual diagramador vai receber esses álbuns
6. Clique em **Importar**. Pronto!

---

## Como distribuir os álbuns para a equipe

Depois de importar, vá na página **Fila** para ver quem está com o quê.

- Cada diagramador aparece com o total de álbuns e em qual etapa estão
- Se quiser **mudar o responsável** de um álbum, clique no nome do diagramador ao lado do álbum e selecione outro

> Só admins (Andrew e Duda) conseguem redistribuir álbuns.

---

## Aviso de código duplicado

Se dois álbuns tiverem o mesmo código (isso acontece na Kaz às vezes), eles aparecem em **laranja** na tela de revisão.

Eles são importados normalmente, mas ficam marcados com o problema **"Formando duplicado"** para você resolver depois. Para resolver: entre no álbum, confira com a Kaz qual é o correto e marque o problema como resolvido.

## O que significa cada status

Os álbuns passam por estas etapas, nessa ordem:

| Status         | O que significa                                                             |
| -------------- | --------------------------------------------------------------------------- |
| **Baixado**    | Chegou no sistema, ainda não foi trabalhado                                 |
| **Descartado** | As fotos foram revisadas e as desnecessárias removidas — pronto para editar |
| **Editando**   | O diagramador está editando                                                 |
| **Montado**    | Álbum montado, aguardando revisão                                           |
| **Enviado**    | Arquivo enviado de volta para a Kaz                                         |
| **Concluído**  | Tudo certo, pagamento confirmado                                            |

> "Descartado" **não significa cancelado**. Significa que a seleção de fotos foi feita e o álbum está pronto para edição.
